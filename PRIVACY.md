# Privacy Policy — Instagram Video Controls Overlay

This extension does not collect, store, or transmit any personal data.

- **No data collection:** The extension only manipulates the HTML5 `<video>` elements already present on the Instagram page you are viewing. It reads no personal information and sends nothing to any server.
- **No storage:** The extension uses no cookies, `localStorage`, or `chrome.storage`. No data persists between sessions.
- **No network requests:** The extension makes no network requests of any kind. All functionality is purely local, in-page JavaScript.
- **No permissions beyond content scripts:** The only capability this extension has is injecting a content script into `https://www.instagram.com/*` pages.

All code is open source and can be reviewed at https://github.com/ChaoZaiOverload/ig-video-controls-overlay.
